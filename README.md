# Radix-Sort-Districts

This project was developed for the Data Structures course.

The program organizes Palestinian districts and towns using linked lists and sorts the districts using the Radix Sort algorithm.

Features
- Doubly Linked List for districts
- Linked List for towns
- Radix Sort for district names
- Sorting towns based on population
- File input/output
- Interactive menu with multiple operations

Programming Language
C
